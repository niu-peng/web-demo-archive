# Web Demo Archive (web-demo-archive)

**描述：** 这是一个收集和展示各种前端技术、Web 概念、算法实现或特定功能演示（Demo）文件的静态资源仓库。所有 `.html` 文件都已通过 GitHub Pages 部署为公共静态资源。

## 🚀 快速访问

您可以通过以下链接直接访问所有部署好的 Demo：

👉 **[Demo 集合主页](https://<你的GitHub用户名>.github.io/<你的仓库名>/)**

*(请注意替换上面的 `<你的GitHub用户名>` 和 `<你的仓库名>` 为你自己的信息)*

## 📂 仓库结构

所有 Demo 均直接存放于仓库根目录或按技术分类存放。

| 目录/文件 | 说明 | 访问示例 |
| :--- | :--- | :--- |
| `index.html` | **（推荐创建）** 仓库的目录页，列出所有可用的 Demo。 | `.../index.html` |
| `/threejs/` | 存放 Three.js、WebGL 相关的 3D 渲染 Demo。 | `.../threejs/cube.html` |
| `/canvas-games/` | 存放基于 Canvas 的小游戏或动画 Demo。 | `.../canvas-games/shooter.html` |
| `*.html` | 其他单一技术的快速演示文件。 | `.../responsive-grid.html` |

## 🛠 如何使用

1. **查找 Demo：** 访问上方的 **Demo 集合主页** 链接，查看 Demo 列表。
2. **直接访问：** 如果您知道 Demo 的文件名，可以直接通过以下 URL 格式访问：
   `https://<你的GitHub用户名>.github.io/<你的仓库名>/<文件名>.html`
